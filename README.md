# SALES-ANALYSIS
This Tableau project shows sales, profit, and product performance using charts and filters. It helps understand which regions and products are doing well. The dashboard makes it easy to analyze data and supports better business decisions.


 Key Features 

Interactive Dashboard:
Built an all-in-one Tableau dashboard to analyze total sales, quantity sold, and customer engagement across different time periods.

Sales Performance Metrics:
Displays key KPIs such as:

YTD Sales: $2.18M

QTD Sales: $811.09K

Products Sold: 27.75K

Customer Reviews: 19.42M

Trend Analysis:
Visualized monthly and weekly sales patterns using line and bar charts, helping identify high-performing periods and seasonal trends.

Category-wise Comparison:
Tabular view showing YTD/QTD sales by product category along with each categoryâ€™s percentage contribution to total revenue.

Top Product Insights:
Highlighted the Top 5 Products based on:

YTD Sales (e.g., Nikon, Atomos, Solid Gear)

YTD Reviews (e.g., SanDisk, JETech, WOLVERINE)

Dynamic Filtering:
Added slicers for Product Type and Quarter Selection to allow quick, customized analysis.

Clean Visual Design:
Used consistent colors, KPIs at the top, and neatly arranged charts to make insights easy to read and presentation-ready.

 Tool Used: Tableau Desktop
 Dataset: Amazon Combined Sales Data (Excel format)
